ubuntu install 谷歌拼音输入法

1 sudo apt-get install fcitx-googlepinyin
2.在settings->Language Support里将keyboard input method system设置为fcitx
3.注销系统，再登录（最好是重启电脑）
在settings->Text Entry里，添加输入源，搜索google-pinyin添加即可
